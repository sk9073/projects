# Ohm Maruthi Orbit - Insurance Management

Ohm Maruthi Orbit is a web application designed to streamline insurance-related tasks. It provides a centralized platform for managing quotes, tracking renewals, and accessing policy details.

## Features

*   **Quote Tracking:** Add, view, and manage insurance quotes. Keep track of details like client name, product, status, company, and quote value.
*   **Renewal Tracking:** Monitor upcoming policy renewals to ensure timely follow-ups.
*   **Policy Details:** Access and manage detailed information about insurance policies.

## Tech Stack

This project is built with the following technologies:

*   **Frontend:**
    *   React
    *   React Router for navigation
    *   Material-UI (MUI) for the user interface components
*   **Backend:**
    *   Firebase for database and authentication
*   **Language:**
    *   TypeScript
