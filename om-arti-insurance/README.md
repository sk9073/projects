## Overview

The Ohm Maruthi Orbit Insurance Management system is a comprehensive CRM-style dashboard built to enhance the operational workflow of insurance agents. It replaces manual tracking with a robust, cloud-synced interface designed for rapid data entry and retrieval.

## Key Features

*   **Quote Tracking:** Add, view, and manage insurance quotes. Keep track of crucial details like client name, product type, current status, provider company, and exact quote value.
*   **Renewal Tracking:** Proactively monitor upcoming policy renewals in a dedicated dashboard to ensure timely client follow-ups and maximize retention.
*   **Policy Management:** Access and manage detailed information about active insurance policies with secure cloud storage.
*   **Authentication & Security:** Secure login and session management to protect sensitive client data.

## Tech Stack

*   **Frontend:** React, React Router
*   **UI Library:** Material-UI (MUI) for consistent, accessible, and responsive components
*   **Backend & Database:** Firebase (Authentication, Firestore Database)
*   **Language:** TypeScript for type safety and scalability
