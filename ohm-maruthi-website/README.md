## Overview

Ohm Maruthi Orbit is a corporate landing page and website designed using Next.js. It features a clean, professional aesthetic tailored for business presentation, emphasizing user experience, fast load times, and SEO best practices.

## Features

*   **Responsive Marketing Pages:** Dynamic and modern sections including Hero, About, Services, and Contact.
*   **Performance First:** Built heavily on React Server Components for optimal loading metrics and SEO scoring.
*   **Custom UI Components:** A cohesive design system providing a premium user interface.
*   **SEO Optimized:** Implementing metadata-first development practices to ensure strong search engine visibility.

## Tech Stack

*   **Framework:** Next.js (React)
*   **Styling:** Tailwind CSS / CSS Modules
*   **Deployment:** Vercel / Firebase App Hosting
